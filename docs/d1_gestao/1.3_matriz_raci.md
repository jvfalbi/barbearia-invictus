# 1.3 Matriz RACI

> **O que é:** matriz que define, para cada pacote da EAP, **quem faz**, **quem aprova**, **quem é consultado** e **quem é informado**.

---

## Legenda

| Sigla | Significado | Quantos por linha? |
|---|---|---|
| **R** | **Responsible** — Quem executa a tarefa | 1 ou mais |
| **A** | **Accountable** — Quem aprova / responde pelo resultado | **Apenas 1** (regra do PMI) |
| **C** | **Consulted** — Quem dá input antes da execução | 0 ou mais |
| **I** | **Informed** — Quem é informado depois de pronto | 0 ou mais |

## Como ler

- Cada **linha** é um pacote de trabalho.
- Cada **coluna** é uma pessoa do projeto.
- Em toda linha deve haver **exatamente um A** (quem responde pelo entregável).
- Em toda linha deve haver **pelo menos um R** (quem coloca a mão na massa).

---

## Como gerar a versão visual

Recomenda-se reproduzir esta tabela em:

- **Google Sheets** ou **Microsoft Excel** com cores condicionais (R = azul, A = verde, C = amarelo, I = cinza)
- Exportar em PDF e anexar à entrega final.

> Pesquisar no YouTube por **"Como preencher uma Matriz RACI"** para ver
> a diferença prática entre quem **faz (R)** e quem **aprova (A)**.
