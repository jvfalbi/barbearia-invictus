## Como exportar os diagramas Mermaid para PNG (1 minuto)

1. Abra o arquivo `1.2_eap.md` ou `1.4_cronograma.md` direto no GitHub (ele renderiza Mermaid nativamente).
2. Dê print da tela ou:
3. Cole o bloco ` ```mermaid ... ``` ` em <https://mermaid.live> e clique em **Actions → PNG/SVG**.
