# 💈 Barbearia Invictus

Sistema desenvolvido para gerenciamento de uma barbearia, oferecendo funcionalidades para controle de clientes, agendamentos e administração dos serviços.

## 🔗 Repositório do Projeto

https://github.com/jvfalbi/barbearia-invictus

---

# 📌 Sobre o Projeto

O projeto **Barbearia Invictus** foi desenvolvido com o objetivo de facilitar o gerenciamento de uma barbearia através de uma plataforma moderna e organizada.

O sistema permite:

- Cadastro de clientes
- Controle de serviços
- Sistema de agendamentos
- Organização administrativa
- Gerenciamento de informações da barbearia
